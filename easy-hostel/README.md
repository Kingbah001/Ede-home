# Easy Hostel

Trust-first student accommodation platform for Federal Polytechnic Ede students only.

## Quick Start

1. Open `index.html` in browser (or `npx live-server .`).
2. Filter/search verified hostels.
3. Click WhatsApp for contact.
4. Use Admin button (bottom-right) to add/edit/verify listings.

## Features

- Verified listings priority (green badges).
- Mobile-responsive.
- LocalStorage persistence.
- Pre-filled WhatsApp messages.

## Edit Data

- `data.js`: Add real hostels.
- Admin panel: Verify, update.

## Success Goal

20+ verified Ede listings, quick finds, real inquiries.
